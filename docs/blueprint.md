# **App Name**: Math Explorers

## Core Features:

- Random Math Problem Generation: Generates random multiplication or division problems based on the specified difficulty levels, ensuring exact divisions.
- Hybrid Input Method: Allows users to input answers using both an on-screen keypad and a physical keyboard.
- Natural Audio Feedback: Uses Text-to-Speech (TTS) to provide natural-sounding audio prompts and feedback, filtering for specific Spanish voices with adjusted pitch and rate.
- Trick System: Provides a modal with visual aids (using CSS/emojis) to help solve math problems, without relying on external images.
- Progressive Web App (PWA) Support: Configured as a PWA for installability and offline access.

## Style Guidelines:

- Background color: Cream/Off-White (#FFF8E1) to minimize visual fatigue.
- Primary color: Soft orange (#FFB74D) for buttons and interactive elements.
- Accent color: Pale yellow (#FFECB3) to highlight correct answers or key features.
- Font: 'Lexend', a sans-serif font, for enhanced readability; base size set to large. Note: currently only Google Fonts are supported.
- Clean, centered layout to reduce distractions and focus attention.
- Celebratory confetti animation (canvas-confetti) on correct answers to provide positive reinforcement.
- Use of 'lucide-react' icons for clear and understandable visual cues.
